# Project Structure

- `src/main.ts`: workspace bootstrap and module wiring.
- `src/modules/`: frontend modules for editor, explorer, chat, docs, contexts, and supporting panels.
- `src/utils/`: shared prompt and utility builders.
- `src-tauri/`: backend filesystem, git, and native command surface.
- `.athva/contexts/`: modular persistent context index, core context files, and task history files.
